# SQL проекты

Проекты в Greenplum:

- `GP_udf_delta_upsert` — создание UDF для загрузки данных по ключу, реализация метода delta upsert.
- `GP_udf_delta_partition` — создание UDF для загрузки данных с помощью подмены партиций, реализация метода delta partition.
- `GP_udf_mart` — создание UDF для сборки и обновления витрины данных, уровень детализации магазин*день, обновление данных осуществляется по методу delta partition.

Проекты в ClickHouse:

- `CH_shards_replicas` — создание реплицированной, распределенной (шардированной) таблиц (масштабирование загружаемых данных).
- `CH_dictionaries` — создание словарей

Проекты в Oracle:

`ORA_proc_time` — функция по определению времени выполнения запроса (фрагмент).

Основной стек: Greenplum, PostgreSQL, Clickhouse, SQLite, MySQL
