# Apache Airflow проекты

- `dag1` — создание DAG для определения последовательности загрузки/обновления таблиц и сборки витрины
- `dag2` — создание DAG для определения последовательности загрузки/обновления таблиц и сборки витрины, настройка запуска по расписанию
- `dag3` — создание DAG для загрузки витрины duckdb в облачное хранилище s3 (фрагмент).

Основной стек: Postgres Operator, Python Operator, TaskGroup
